ASÍ SE DESARROLLA WINDOWS 98

Por Valentina Leiva | Junio 1997

Windows 95 llegó al mercado como un león, con millones de dólares invertidos en publicidad y una campaña en los medios fuera de control. La llegada de Windows 98, el reemplazo de Microsoft para el sistema anterior, llegará con un estilo más parecido al de un cordero.

ESPECIAL PARA EL TIEMPO

Windows 95 representaba una mejora genuina y drástica del software básico utilizado para controlar casi todos los computadores personales en los hogares del mundo. Lucía y funcionaba de una manera muy diferente a como lo hacía su predecesor, Windows 3.1.

Por el contrario, Windows 98 luce y funciona de una manera muy similar a Windows 95. Así, la llegada de este nuevo producto (programada en forma tentativa para el segundo trimestre de este año, algún día entre abril y junio), no tendrá el encanto que tuvo el lanzamiento de la versión previa.

Microsoft ha adoptado una estrategia más discreta en esta ocasión, debido principalmente a los problemas por violación de las leyes antimonopolio que lo enfrenta con el Departamento de Justicia de Estados Unidos.

Pero, a pesar de lucir parecido a Windows 95, Windows 98 contiene cambios internos fundamentales que pueden convertirlo en la versión más importante de este sistema operativo jamás desarrollada.

Estas son, a vuelo de pájaro, las características más llamativas del nuevo sistema de Microsoft:

Incluso en la versión beta (o de prueba), que es utilizada por cientos de miles de usuarios de computadores personales en estos momentos, Windows 98 es mucho más estable que su precursor. Parece ser menos propenso a caerse.

Funciona mejor al reconocer nuevos accesorios, como módem, tarjetas de sonido y tarjetas de video. Además, aprovecha mejor las ventajas de un cambio en los equipos llamado USB (por la sigla en inglés de bus serial universal, el sistema interno de comunicaciones de la máquina), que les permite a los usuarios conectar, literalmente, cientos de dispositivos, sin preocuparse por conflictos de hardware que puedan paralizar el sistema.

De hecho, con el USB, un dispositivo puede ser conectado o removido sin necesidad de apagar el computador y reiniciar el sistema.

Tiene en cuenta el deseo de la mayoría de los usuarios de equipos, con respecto a las velocidades de acceso a Internet.

Con Windows 98 es posible conectarse más rápido, incluso sin necesidad de un módem de cable (para redes de fibra óptica) o de líneas RDSI.

El sistema operativo tiene una característica que les permite a los usuarios que tienen más de una línea telefónica utilizar un módem independiente para cada una de ellas. Ambos dispositivos se conectan con el proveedor de acceso, de manera que en conjunto incrementan la velocidad de transmisión.

También con respecto a Internet, Windows 98 ofrece una tecnología bastante exótica. El sistema le permitirá al computador obtener información de las ondas que viajan por el aire, a gran velocidad.

A pesar de que esta tecnología todavía no está en uso (de hecho no se ha previsto una fecha para su lanzamiento al mercado), las estaciones de televisión podrán enviar datos junto con las emisiones de sus programas regulares.
